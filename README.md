# My Python Learning Journey

这个仓库包含我本学期写过的、最满意的 Python 程序之一：零食投票统计系统
snack_vote()。\
它可以根据用户输入的票数，自动统计总票数、最高票、最受欢迎零食，并生成排名列表。

------------------------------------------------------------------------

项目功能简介

`snack_vote()` 函数可以完成以下任务：

-   输入四种零食的票数\
-   自动计算总票数\
-   找出最高票及对应该零食\
-   如果有并列，会自动显示多个最受欢迎零食\
-   根据票数排序并显示百分比

代码说明

以下是该程序的核心逻辑：

``` python
def snack_vote():
    # 统计零食和对应的票数
    snacks = {
        '爆米花': int(input("爆米花票数：")),
        '薯片': int(input("薯片票数：")),
        '巧克力': int(input("巧克力票数：")),
        '紫菜': int(input("紫菜票数："))
    }

    # 统计票数
    total = sum(snacks.values())
    max_votes = max(snacks.values())
    favorite = [s for s, v in snacks.items() if v == max_votes]

    # 输出统计结果
    print(f"\n总票数：{total}")
    print(f"最高票：{max_votes}")
    print(f"最受欢迎：{'、'.join(favorite)}")

    print("\n排名：")
    for i, (s, v) in enumerate(sorted(snacks.items(), key=lambda x: x[1], reverse=True), 1):
        percentage = (v / total) * 100
        print(f"{i}. {s} {v}票 ({percentage:.1f}%)")
```
